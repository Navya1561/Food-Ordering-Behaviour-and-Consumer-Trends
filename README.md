# Food-Ordering-Behaviour-and-Consumer-Trends
# 🍽️ Food Ordering Behaviour Analysis

## 📌 Project Overview
This project analyzes customer food ordering behaviour using Tableau dashboards and presents the insights through a Flask-based web application.

## 🚀 Features
- Interactive Tableau Dashboard
- Data Visualization
- Responsive Bootstrap Website
- Flask Backend
- Customer Behaviour Analysis

## 🛠️ Technologies Used
- Python
- Flask
- HTML
- CSS
- Bootstrap
- Tableau Public

## 📂 Project Structure
```
project/
│── app.py
│── templates/
│   └── index.html
│── static/
│   ├── css/
│   ├── js/
│   ├── img/
│   └── vendor/
```

## ▶️ How to Run
1. Install Flask
   ```
   pip install flask
   ```
2. Run the application
   ```
   python app.py
   ```
3. Open the browser
   ```
   http://127.0.0.1:5000
   ```

## 📊 Dashboard
The website contains an embedded Tableau dashboard for visualizing food ordering behaviour.
